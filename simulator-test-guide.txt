================================================================================
                        시뮬레이터/에뮬레이터 테스트 가이드
================================================================================

--------------------------------------------------------------------------------
iOS 시뮬레이터
--------------------------------------------------------------------------------

# 사용 가능한 시뮬레이터 목록 보기
xcrun simctl list devices available

# 특정 기기로 실행
npx expo run:ios --device "iPhone SE (3rd generation)"
npx expo run:ios --device "iPhone 16 Pro"
npx expo run:ios --device "iPad mini (6th generation)"

# Expo Go로 실행 (선택 화면 나옴)
npx expo start --ios


--------------------------------------------------------------------------------
Android 에뮬레이터
--------------------------------------------------------------------------------

# 사용 가능한 에뮬레이터 목록 보기
emulator -list-avds

# 에뮬레이터 먼저 실행
emulator -avd <에뮬레이터_이름>

# 앱 실행
npx expo run:android

# Expo Go로 실행
npx expo start --android


--------------------------------------------------------------------------------
간단하게 (대화형)
--------------------------------------------------------------------------------

# 개발 서버 시작
npx expo start

# 키보드 단축키:
#   i         → iOS 시뮬레이터 (기본)
#   a         → Android 에뮬레이터 (기본)
#   shift + i → iOS 기기 선택
#   shift + a → Android 기기 선택


--------------------------------------------------------------------------------
테스트 체크리스트
--------------------------------------------------------------------------------

[ ] iPhone SE (작은 화면)
[ ] iPhone 14 Pro / 16 Pro (기준 화면)

[ ] 폰트/간격 - 화면 크기에 따라 적절히 스케일링
[ ] 앱 크래시 없음
